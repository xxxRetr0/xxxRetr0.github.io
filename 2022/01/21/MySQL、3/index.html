<一>、这里是一些小东西

1、select * from 表名； 查询这个表里的所有数据

2、desc 表名； 只看表的结构，不看表的内容

3、select version(); 查MySQL版本

4、select database(); 查看当前使用数据库

5、不见分号不停止
mysql> show
->
->
-> databases
-> ;
+————————+
| Database |
+————————+
| bjpowernode |
| information_schema |
| mysql |
| performance_schema |
| sys |
+————————+
5 rows in set (0.00 sec)

6、用 \c 停止一条命令的输入，或直接用 ； 来报错停止
mysql> show
->
->
-> \c

7、SQL语句的分类

DQL：(Data Query Language)
        数据查询语言
        select.......

DML：(Data Manipulation Language)
        数据操作语言（对于表中内容的增删改）
        insert...    
        delete...    
        update...
        针对操作表中的数据data

DDL：(Data Definition Language)
        数据定义语言（对于表的结构的增删改）
        create...    
        drop...    
        alter...
        针对操作表的字符段

TCL：(Transactional Control Language)
       事务控制语言（对于事务的提交、回滚）
       commit...
       rollback...

DCL：(Data Control Language)
       数据控制语言
       授权grant...
       撤销权限revoke...

<二>、DQL

select语句只做检索，不做任何修改，无大小写要求

一、简单查询（楞查）

1、查询一个/多个/所有字段
select 字段名 from 表名；
select 字段名,字段名 from 表名；
select * from 表名；
select * from 表名；一般用于临时查看表内容，因计算机运行时还有把 * 转化为 a,b,c,d,… 的过
程，若要用于程序内，最好用 select a,b,c,d,… from 表名； 有更强的可读性和运行效率

2、起别名
select 原名 as 别名 from 表名；
as可省略–>select 原名 别名 from 表名；
只在下面的显示处为改后的名字，原名字还是一样，没有修改
若别名有空格或中文，需用 单/双引号 将整个别名括起来，不然会报错，其中双引号只在mysql
可用，其他数据库用不了，单引号才是标准

3、列参与数字运算
列名可直接参与运算，如 select ename,sal*12 from emp;

mysql> show * from dept;
mysql> select ename,sal from emp;
mysql> select ename,sal*12 as ‘年薪’ from emp;

二、条件查询（筛选）

格式：select 字段1,字段2，字段3，… from 表名 where 条件；

= 等于

！=和<> 不等于

,<,>=,<= 大于，小于，大于等于，小于等于

between … and …和>= and <= 两者之间，其中between必须从小到大，且为闭区间

is null 和 is not null 为空 和 不为空，其中 is 不能用 =，且 0 和 null 为两码事

and 和，等价于C的&&， or 或，等价于C的||，其中 and 的优先级高于 or

in（） 可理解为多个 or ，括号内为具体的值或者词，而不是一个开区间类似的还有 not in（）

mysql> select empno,ename from emp where sal = 800;
mysql> select empno,ename from emp where sal != 800;
mysql> select empno,ename from emp where sal <> 800;
mysql> select empno,ename，sal from emp where ename = ‘SMITH’;
mysql> select empno,ename from emp where sal >= 2450 and sal <= 3000;
mysql> select empno,ename from emp where sal between 2450 and 3000;
mysql> select * from emp where sal >2500 and (deptno = 10 or deptno = 20);
mysql> select ename,sal from emp where job in(‘MANAGER’ or ‘SALESMAN’);
mysql> select ename,sal from emp where sal in (800,5000);

三、模糊查询

like ，表示模糊查询，与 % ， _ 两者相匹配，其中 % 可用于表示任意多个字符， _可用于表示
任意一个字符

mysql> select ename from emp where ename like ‘%A%’;–>查名字里有A的人，同样’%A’表
示以A结尾，’A%’表示以A开头，’_A%’名字第二个字母为A，其他同理

%和_本身就有特殊含义，若要查询的内容中就有%或者_，在二者前加 \ 即可，如’%_%’
其中 \ 称为转义字符

四、排序

1、单个字段排序

mysql> select ename,sal from emp order by sal; 默认升序
mysql> select ename,sal from emp order by sal asc; 指定升序
mysql> select ename,sal from emp order by sal desc; 指定降序

2、多个字段排序

mysql> select ename,sal from emp order by sal asc,ename asc;//sal起主导作用，只有当sal相
同时才会启用ename排序

3、结合上面的来一波

mysql> select ename,sal from emp where sal between 1250 and 3000 order by sal desc;

我们输入时严格按照顺序：select … from … where … order by …
而计算机执行的顺序为 from–>where–>select–>order by
