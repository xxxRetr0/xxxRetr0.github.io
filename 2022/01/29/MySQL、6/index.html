一、创建表

1、建表语句的语法格式：
create table 表名(字段名1 数据类型,字段名2 数据类型,字段名3 数据类型,….);
表名一般以 t_ 或 tbl_ 开始

2、MySQL中的数据类型

    int        整数型，最长11
    bigint                    长整型
    float        浮点型
                            double                    双精度浮点型数据
    char        定长字符串，最长255
                                        若给定10个字符，而输入了4个字符，则显示即为4个字符加6个空格不管实际长度，分配固定的空间用于存储数据

    varchar                    可变长字符串，最长255
                                        若给定10个字符，而输入了4个字符，则显示即为4个字符，根据实际数据长度动态的分配存储空间
                                        
    date        短日期类型
                            datetime                 长日期类型
    blob        二进制大对象（存储图片、视频等流媒体信息） Binary Large OBject 
    clob        字符大对象（存储较大文本，最多可以存储4G的字符串。） Character Large OBject（
    ......

3、char 与 varchar
char 速度快，但使用不当容易浪费空间，varchar 速度慢，但节省空间
选择：给性别等固定长度的用 char，给姓名等不定长度的用 varchar

4、删除表：①drop table 表名；//如果该表本就不存在，会报错
②drop table if exists 表名；

mysql> create table t_student(no int,name varchar(32),gender char(1),age int(3),email varchar(255));

二、插入数据 insert（DML）

1、格式：
insert into 表名（字段1，字段2，字段3…） values（值1，值2，值3…）；

  字段名和值的数量和数据类型要一一对应

mysql> insert into t_student(no,name,gender,age,email) values(1,’zhangsan’,’m’,’20’,‘zhangsan@123.com‘);
mysql> insert into t_student(email,name,gender,age,no) values(‘lisi@123.com‘,’lisi’,’f’,20,2);

2、指定默认值（default）

表中的 default 为默认值，可在创建表的时候给定默认值

mysql> create table t_student(no int,name varchar(32),gender char(1) default ‘m’,age int(3),email varchar(255));

插入数据时：
①没给的数据会附上默认值
②表名后的字段可省略，此时后面的值个数必须写全，否则报错，输入的值必须和默认的字段一一对应

3、格式化数字：format(数字,’格式’);

mysql> select ename,format(sal,’$999,999’) as sal from emp;

4、转整型为日期型
str_to_date(‘字符串日期’,’日期格式’)

mysql日期格式：%Y  年
                           %m 月
                           %d  日
                           %h  时
                           %i   分
                           %s   秒

如：‘01-10-1990’用 str_to_date(‘01-10-1990’,’%d-%m-%Y’)
若我们输入的日期格式直接为 ‘%Y-%m-%d’ 这一类，则不用 str_to_date 函数

5、日期格式化
data_format(日期类型数据,’日期格式’)

常用于查询日期时，设置日期的格式

mysql> select id,name,date_format(birth,’%d/%m/%Y’) birth from t_user;

三、时间

MySQL短日期默认格式：%Y-%m-%d

MySQL长日期默认格式：%Y-%m-%d %:h%:i%:s

now()：显示当前时间，数据类型为datetime

新建个表 mysql> create table t_user(id int,name varchar(32),birth date,create_time datetime);
mysql> insert into t_user values(1,’zhangsan’,’1990-10-01’,’2022-1-29 22:40:30’);
mysql> insert into t_user values(2,’lisi’,’1990-10-01’,now());

四、修改数据 update（DML）

格式：
update 表名 set 字段名1=值1,字段名2=值2,字段名3=值3… where 条件;

    注意：没有条件整张表数据全部更新。

mysql> update t_user set name = ‘jack’,birth = ‘2000-2-2’ where id = 2;

五、删除数据 delete（DML）

格式：
delete from 表名 where 条件;

     注意：没有条件全部删除。

删除大表中的数据？
truncate table 表名; // 表被截断，不可回滚。永久丢失。

mysql> delete from t_user where id = 2;//删除第二行👇①
mysql> delete from t_user;//删除全表👇👇②
